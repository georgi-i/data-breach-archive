# Bulgarian Data Leaks Archive

This repository is dedicated to preserving intelligence about Bulgarian data leaks published online. It serves as an open reference to document significant breaches impacting various Bulgarian organizations, companies, and government entities. The information herein is intended for cybersecurity professionals, researchers, and interested parties to track, analyze, and respond to these breaches.

## Purpose

Data breaches pose serious risks to individuals and institutions by exposing sensitive information. This archive aims to collect and maintain a record of known Bulgarian data leaks to support transparency, awareness, and defensive measures within the cybersecurity community.
